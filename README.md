# 💜 Project X – A Surprise Birthday Game Experience 🎁

**Project X** is a JavaFX-based, multi-checkpoint game experience handcrafted with love and creativity. Designed as a surprise birthday gift, the game unfolds through a series of challenges, visuals, and heartwarming messages, culminating in a personalized celebration.

---

## 🎮 Game Structure

The game is divided into **three checkpoints**, each unlocking sequentially:

### ✅ Checkpoint 1 – Wordle Challenge
A 5-letter word guessing game inspired by the classic **Wordle**. Type in your guesses and hit **Enter**. Hints become progressively more visible after failed attempts.

### ✅ Checkpoint 2 – Snakes & Ladders
A playful board game with a twist:
- Arrows represent snakes 🐍 and ladders 🪜.
- Dynamic dice rolls and grid animations.
- Beautiful custom background imagery and styled tokens.

### ✅ Checkpoint 3 – Final Tiles Puzzle
A **3x3 tile grid** hiding the final celebration. Only one tile is correct – choose wisely! The rest will flip to surprise visuals 💐.

## ▶️ Running the Project

### Using JDK 17 and JavaFX:
```bash

```
